# Events timeline for context 

## Deployment

1. Clone this repo ```git clone https://github.com/kevinboudreault/timeline-viewer.git```
2. update docker-compose.yml [port/main-key]
3. execute ```RAILS_MASTER_KEY={mastey.key content} docker compose up --build -d```

## Purpose

I created this app to be able to have a view of events [email/sms/phone/irl] with a timeline context to keep track of ongoing issues and follow-up.
